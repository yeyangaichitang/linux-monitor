## 关键命令速查

| 用途 | 命令 |
|------|------|
| 查看服务状态 | systemctl status nginx |
| 启停服务 | systemctl start/stop nginx |
| 开机自启 | systemctl enable nginx |
| 防火墙放行端口 | firewall-cmd --add-port=80/tcp --permanent && firewall-cmd --reload |
| 查看系统日志 | journalctl -u nginx |
| 查看错误日志 | tail /var/log/nginx/error.log |
| 检查端口监听 | ss -tlnp \| grep :80 |
| Git 推送 | git add . && git commit -m "备注" && git push |
